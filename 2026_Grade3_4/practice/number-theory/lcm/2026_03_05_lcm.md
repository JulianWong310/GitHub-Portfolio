# 🎓 Math & Code: The LCM-GCD Connection

### 1. The Definitions
* **GCD (Greatest Common Divisor):** The biggest number that divides both $a$ and $b$.
* **LCM (Least Common Multiple):** The smallest number that both $a$ and $b$ can multiply into.

---

### 2. The Property of LCM

Any **common multiple** of two numbers $a$ and $b$ must be a **multiple of their LCM**.

> **In simple terms:** If $M$ is a common multiple of $a$ and $b$, then $LCM(a, b)$ can perfectly divide $M$.
> $$LCM(a, b) \mid M$$

---

###  Example 
Let’s use $a = 2$ and $b = 5$:
* **LCM(2, 5) = 10**
* **Other Common Multiples:** 20, 30, 40, 50...
* **The Connection:** Notice that 20, 30, 40, and 50 are all **multiples of 10**! 
* You cannot find a common multiple of 2 and 5 (like 15 or 25) that 10 cannot divide.

---

### **✍️ Quick Test**
Given $\text{lcm}(a, b) = 15$:
* Is $45$ a common multiple? **Yes**, because $15 \mid 45$.
* Is $50$ a common multiple? **No**, because $15 \nmid 50$.

### **💻 Python Proof Snippet**
```python
# Theorem 1 Check
L = lcm(a, b)
M = a * b * 3  # M is definitely a common multiple
print(M % L == 0) # This will ALWAYS be True
```

### 3. The Magical Formula
For any two positive integers $a$ and $b$:
$$LCM(a, b) = \frac{a \times b}{GCD(a, b)}$$

---

### The "Lego" Proof (Why it works)
Imagine every number is built from **Prime Factor** blocks. 
Let's use $a=12$ and $b=18$:

* **Blocks for 12:** $\{2, 2, 3\}$
* **Blocks for 18:** $\{2, 3, 3\}$

#### The Discovery:
1.  **The Shared Blocks (GCD):** Both numbers share one `2` and one `3`. 
    * $GCD = 2 \times 3 = 6$.
2.  **The Multiplication ($a \times b$):** If you multiply them, you get ALL the blocks: 
    * $\{2, 2, 3\}$ + $\{2, 3, 3\} = \{2, 2, 2, 3, 3, 3\}$.
    * *Notice:* The shared blocks ($2, 3$) appear **twice**!
3.  **The LCM:** To build the LCM, we only need the shared blocks **once**.
    * **Logic:** $LCM = \frac{\text{All Blocks}}{\text{Shared Blocks Once}}$


---

### 4. Python Implementation
We can calculate LCM using his existing GCD function:

```python
def gcd(a, b):
    while b!=0:
        a, b = b, a % b
    return a

def lcm(a, b):
    # Apply the Magical Formula
    return (a * b) // gcd(a, b)

# Test it!
print(lcm(12, 18))  # Output: 36
```
